# CoCo Skills

Reusable Snowflake Cortex Code skills for data engineering, migration, modelling, and solution acceleration.

## Skills

- `dv2-to-dimensional`: Convert Snowflake Data Vault 2.0 models, downstream `.dv.sql` queries, and dbt Data Vault projects into Kimball-style dimensional designs.

## Usage

Clone this repository and copy the required skill folder into your Cortex Code skills directory, or register the repository as a custom skill source where supported.
